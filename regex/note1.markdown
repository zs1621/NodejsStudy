##**正则元字符**

 - `?` -> 匹配0次或者1次
  - `colou?r` 同时匹配 color 和 colour:w

 - `*` -> 匹配0次或者多次
  - `ABC\d*` 会匹配ABC, ABC1, ABC12, ...

 - `+` -> 匹配一次或者多次前面的块, 至少匹配一次
  - `ABC\d+` 会匹配ABC1, ABC12..但不匹配ABC

 - `{n}` -> 重复大括号前面n次
  - `ABC\d{3}` 匹配 ABC123, *ABC123*22斜体部分

 - `{n, m}` -> 至少重复匹配大括号前面的n次，最多m次(n<=m)
  - `A\d{2, 3}` 匹配A12, A123, A23313413413, ...

 - `{n, }` -> 至少匹配n次，至多无限匹配无限次
